# Warsh Mushaf Fonts

This directory contains fonts for the Warsh Mushaf (North African riwaya) support.

## Required Fonts

The following font files should be added here:

### Primary Mushaf Font
- `WarshQuran-Regular.ttf` - Regular weight for main Quranic text
- `WarshQuran-Bold.ttf` - Bold weight for emphasis

### Secondary Text Font
- `WarshText-Regular.ttf` - Regular weight for UI text and translations
- `WarshText-Medium.ttf` - Medium weight for headings
- `WarshText-Bold.ttf` - Bold weight for titles

## Font Sources

Warsh Mushaf fonts can be obtained from:

1. **KFQPC Fonts** - King Fahd Quran Printing Complex fonts
2. **Amiri Font** - Open source Naskh font (can be adapted)
3. **Scheherazade New** - Open source font with good Arabic support

## Technical Notes

- Warsh riwaya has different diacritical marks compared to Hafs
- The fonts should support all Arabic Unicode blocks used in the Quran
- Ensure proper OTL (OpenType Layout) features for Quranic text
