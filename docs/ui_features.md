# Premium UI Features Documentation 🎯🚀🔥

This document details the premium UI features added to the `imad_flutter` library during the Etqan Ramadan Initiative.

## App Branding
- **BrandedSplash**: A custom, animated splash screen for a professional app startup.
- **PremiumLogo**: A flexible logo widget featuring the "Etqan Mushaf" identity.
- **LibraryHome**: A refreshed home page layout for the example application using `SliverAppBar`.

## Reading Themes
- **AMOLED Theme**: A specialized high-contrast, pure-black theme for OLED screens.
- **Theme Scoping**: All components automatically adapt to the active `ReadingTheme`.

## Search Enhancements
- **Animations**: Results populate with smooth fade and slide transitions.
- **Highlighting**: Accurate, Arabic-aware query highlighting for surah names and verses.

## Quran Viewer Improvements
- **Premium Header**: Displays Juz, Chapter, and Page in a clean, organized layout.
- **Adaptive Toolbar**: A smart toolbar button that manages its own visibility and opacity based on user interaction.
- **Premium Index Header**: A beautifully designed header for the chapter selection drawer.
- **Verse-Level Highlighting**: Supports precise selection and audio synchronization by highlighting verse regions on the page image.
- **Verse Markers (Fasel)**: Renders beautiful, numbered verse separators at the exact positions where verses end.

## Analytics & Tracking
- **ReadingHistoryPage**: A dedicated dashboard for tracking reading statistics (streaks, time read, pages read) and resuming previous sessions.

## Audio Controls
- **Navigation**: Support for skipping to the next or previous chapter directly from the audio bar.
- **Privacy**: User preference to toggle the visibility of the audio player bar.
